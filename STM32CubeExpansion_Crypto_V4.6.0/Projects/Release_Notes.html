Release Notes for Projects of X-Cube-CRYPTOLIB Package

Copyright © 2005 STMicroelectronics

Purpose

STM32 Cryptographic package provides a rich set of examples covering the main features of each cryptographic algorithm. All examples are available in source file format and provided with preconfigured projects for the supported board. Expected output data are added in almost examples in order to check that the examples work correctly. Expected data are taken from cryptographic standard. Please refer to readme.txt in each example for more details about cryptographic standard used.

The exhaustive list of projects is provided in this table: STM32CubeProjectsList.
